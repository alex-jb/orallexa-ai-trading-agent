## Color Reference (for social/marketing)

When creating new assets, use the DESIGN.md color palette:

| Token | Hex | Visual |
|-------|-----|--------|
| Gold (primary) | `#D4AF37` | Brand accent, CTAs |
| Gold bright | `#FFD700` | Highlights, gradients |
| Gold muted | `#C5A255` | Secondary accents |
| Background | `#0A0A0F` | Dark base |
| Emerald (bull) | `#006B3F` | Buy signals, success |
| Ruby (bear) | `#8B0000` | Sell signals, errors |
| Champagne (text) | `#F5E6CA` | Primary text |

### Brand Gradient
```
linear-gradient(135deg, #D4AF37, #FFD700, #C5A255)
```
